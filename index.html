PROMPT GENERATOR for SUNO AI

START

사용 방법

  1. START 버튼 클릭: 프롬프트 생성 페이지로 이동합니다.
  2. 옵션 선택: 좌측에서 가사 및 음악 스타일 관련 옵션을 선택하거나 입력합니다.
  3. 프롬프트 생성 확인: 우측에서 생성된 'Lyrics Prompt'와 'Music Style Prompt'를 확인합니다.

Suno AI 적용 단계

  1. 가사 생성용 프롬프트 활용:
    • 생성된 'Lyrics Prompt (가사 생성용)'를 복사합니다.
    • 복사한 프롬프트를 Gemini Pro 2.5와 같은 고성능 언어 모델에 붙여넣어 가사를 생성합니다.
    • Gemini가 생성한 가사를 복사합니다.
    • 복사한 가사를 Suno AI의 가사(Lyrics) 입력창에 붙여넣습니다.
  2. 음악 스타일 프롬프트 활용:
    • 생성된 'Music Style Prompt'를 복사합니다.
    • 복사한 프롬프트를 Suno AI의 음악 스타일(Style of Music) 입력창에 붙여넣습니다.
  3. Suno AI 생성: Suno AI에서 노래 생성을 시작합니다.

프롬프트 저장: 필요시 생성된 프롬프트를 'Save Lyrics' 또는 'Save Style' 버튼을 눌러 텍스트 파일로 저장할 수 있습니다.

How to Use

  1. Click START Button: Navigates to the prompt generation page.
  2. Select Options: Choose or input desired options in 'Lyrics Prompt Options' and 'Music Style Prompt Options' on the left.
  3. Check Generated Prompts: Review the 'Lyrics Prompt' and 'Music Style Prompt' generated on the right.

Applying to Suno AI

  1. Using the Lyrics Prompt:
    • Copy the generated 'Lyrics Prompt (for Lyrics Generation)'.
    • Paste the copied prompt into a high-performance language model like Gemini Pro 2.5 to generate lyrics.
    • Copy the lyrics generated by Gemini.
    • Paste the copied lyrics into Suno AI's Lyrics input field.
  2. Using the Music Style Prompt:
    • Copy the generated 'Music Style Prompt'.
    • Copy the copied prompt into Suno AI's Style of Music input field.
  3. Generate in Suno AI: Start the song generation process in Suno AI.

Saving Prompts: If needed, you can save the generated prompts as text files using the 'Save Lyrics' or 'Save Style' buttons.
